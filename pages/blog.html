Blog folder structure

How to add a new post

Create a new folder under assets/blogs/ using the pattern W0001 for work or L0001 for life posts.

  • blog_id.txt — short summary or post metadata
  • blog_id.html — the actual post content page
  • assets/blog_id_xxx.png — featured image or related illustration

Duplicate blog-template.html to start a new post, then link to it from this listing page.
